## How to Use

1. **Selection**: Choose a canvas specification from the dropdown menu that matches your project size.
2. **Import**: Use the "Load Image" button to select your reference file.
3. **Alignment**: Use "Move" and "Resize" modes to align the overlay with your drawing area.
4. **Lock**: Click "Lock Position" to freeze the overlay.
5. **Filter**: Click on any color code in the right-hand list to highlight its specific locations on the grid.

## System Requirements

- **Operating System**: Windows 10 or 11
